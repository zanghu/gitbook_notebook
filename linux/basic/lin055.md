## Linux: 查看指定路径的挂载点

* **需求**

  日常运维经常需要检查剩余磁盘空间大小，以便评估磁盘空间不足问题发生的可能性。

* **困难**

  `df-h`命令虽然能够列出当前环境各个设备的空间大小信息，但需要先正确判断出当前用户正在使用的那些路径（比如主目录）位于哪个设备（挂载点）下。

  ![](/assets/lin055_01.png)

* **方法**

  使用如下命令查询指定路径的挂载点

  ```shell
  # 查询/home/zanghu所在的挂载点
  $ df -h /home/zanghu
  ```
