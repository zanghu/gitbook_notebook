## Linux: 创建新用户

### 1.创建用户

添加一个新用户，注意执行下面的操作前应先`su -`切换至root账户，如果使用`sudo`执行下面的命令，那么创建出的目录都是只有sudo用户才能修改。

```shell
# 切换至root
$ su -

# -d指明用户主目录
# -m指示自动创建
# 最后的jiayq是用户名
$ useradd -d /home/jiayq -m jiayq
```

### 2.配置密码

没有配置密码的用户无法通过普通用户切换，只能通过root用户切换，因此应尽快为新用户配置密码。

```shell
# passwd <用户名>
$ passwd jiayq
```
