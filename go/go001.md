### 1.项目目录

项目目录是demo，目录下文件如下图

![](/assets/go001_01.png)

`package main`中调用了`package cfg`，文件源码如下图，注意包之间的调用关系

**文件**：main.go

```go
// main.go
package main

import (
    "fmt"
    "os"
    "app/cfg"
)

func main() {
    cfg.Test()
    fmt.Fprintf(os.Stdout, "Hello\n")
}
```

**文件**：cfg/test.go

```go
// cfg/test.go
package cfg

import "fmt"

func Test() {
    fmt.Println("test")
}
```

### 2.编译构建

#### 2.1.使用`go mod`

在`demo`目录下执行：

```shell
$ go mod init app
```

![](/assets/go001_03.png)

这里app是自定义的模块名，执行完成后demo目录下会生成`go.mod`文件

![](/assets/go001_02.png)

#### 2.2.执行go build

在`demo`目录下执行：

```go
$ go build
```
