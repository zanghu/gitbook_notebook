### 1.极简热力图

```python
import numpy as np
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
 
sns.set()
np.random.seed(0)
uniform_data=np.random.rand(10,12)
ax=sns.heatmap(uniform_data)
plt.savefig('heat_map.png')
plt.show()
```
