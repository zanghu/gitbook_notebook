## C: libdl中的函数：dlopen、dlclose、dlsym等等

C程序在运行期加载动态库，一个比较灵活的办法就是使用libdl库中的`dlopen`函数，在程序运行期“打开”指定的动态库。

### 1.例子

测试程序有三部分组成：

* 动态库源码
* 测试用例源码
* 编译脚本

#### 动态库源码

calculate.c

```c
int add(int a,int b)
{
    return (a + b);
}

int sub(int a, int b)
{
    return (a - b);
}

int mul(int a, int b)
{
    return (a * b);
}

int div(int a, int b)
{
    return (a / b);
}
```
