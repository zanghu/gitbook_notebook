## C: valgrind常见用法总结

[valgrind官方网站](http://valgrind.org/)

### 1.下载安装

目前最新版本是3.13.0

```shell
$ tar -jxvf valgrind-3.13.0.tar.bz2
$ cd valgrind-3.13.0
$ ./configure --prefix=/path/to/your/install/dir
$ make all -j8
$ make install
```

修改`~/.bashrc`文件，配置`PATH`环境变量，之后

```shell
$ source ~/.bashrc
```
